Installation#

git clone git@github.com:MBSck/p2obp.git
cd p2obp
pip install -e .

Optional#

An optional dependency is R. van Boekel’s calibrator_find.pro. This is an IDL software used for calibrator finding, from which night_plans can be manually created.

These are then used to feed the parse module of p2obp used within the create_obs (see Features).
